# Ngôn ngữ ink! là gì?
+ Ink! là ngôn ngữ lập trình smart contract trên hệ sinh thái Polkadot 
+ Do đội ngũ `Parity` maintain
+ Build ra wasm 

+ Ink! = Rust + macros

![](../assets/ink_rust.png)
